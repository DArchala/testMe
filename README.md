# TestMe

## Showcase of the application
 soon

## Functionalities of TestMe application
- logging  in
- registering 
- creating new exams
- taking exams
- editing exams
- displaying, sorting, paging exam attempts
- editing user role
- deleting users
- displaying, sorting, paging users
- password resetting
- password changing

## Tools and technologies
- Java
- Angular
  - Typescript
  - HTML
  - CSS
- Hibernate
- H2 Database
- Spring Boot
- Spring Security
- Spring Data
- JUnit
